# NextGen Auth - Login & Signup Form

A modern, responsive, and aesthetically pleasing Authentication UI built with HTML, CSS, and Vanilla JavaScript. This project features a clean design with smooth animations and form validation.

## ✨ Features

- **Modern UI/UX**: Designed with the 'Inter' font family and a soft gradient background for a premium feel.
- **Responsive Design**: Fully functional on mobile, tablet, and desktop screens.
- **Form Validation**: Real-time validation for password matching on the signup page.
- **Interactive States**: Hover effects, focus rings, and active states for buttons and inputs.
- **Smooth Animations**: Subtle fade-in animations for a polished user experience.
- **Loading Simulation**: A simulated "Signing in..." state on the login form to mimic real-world behavior.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure for better accessibility and SEO.
- **CSS3**: Custom properties (variables), Flexbox, and Keyframe animations.
- **JavaScript**: Client-side logic for form submission and validation.
- **Google Fonts**: 'Inter' for modern typography.

## 📁 Project Structure

```text
auth-form/
├── index.html      # Login page
├── signup.html     # Registration page
├── style.css       # Global styles & design system
└── script.js       # Form logic & validation
```

## 🚀 How to Run

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Switch between Login and Signup pages using the links provided at the bottom of each form.
